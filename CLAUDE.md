# ブルーアーカイブ TL記録支援ツール

ブルーアーカイブの総力戦・大決戦等でスキル使用タイミングを記録・共有するためのタイムラインツール。
React + Vite + TypeScript で構築。

## 技術構成

- **フレームワーク**: React 18 + TypeScript
- **ビルド**: Vite
- **状態管理**: useReducer（外部ライブラリ不使用）
- **D&D**: HTML5 Drag and Drop API（ライブラリ不使用）
- **目盛り描画**: Canvas（DOM要素だとグリッド数が多すぎるため）

## 生徒データ

- ストライカー: `data/characters_st.json`
- スペシャル: `data/characters_sp.json`
- 形式: `{ name: string, image: string }[]`
- 画像は GitHub 上の顔アイコン URL
- 共有コーデックで名前→画像URLの逆引きにも使用

## 画面構成

### 生徒登録パネル（上部）
- STRIKER 4枠 + SPECIAL 2枠の計6スロット
- **編集モード / 使用モード** のトグル切替
  - 編集モード: 生徒の追加・変更（クリック→検索モーダル）・削除（×ボタン）が可能。ドラッグ不可
  - 使用モード: ドラッグ&ドロップのみ可能。×ボタン非表示、クリック変更無効
- 初回は編集モードで開始
- **スキルコスト入力**: 各スロット下に数値入力（デフォルト3、0.5刻み）
- **固有4チェックボックス**: SPスロットのみ表示、コスト上限+0.5

### タイムライン（中央）
- 左が最大時間、右が0:00のカウントダウン方向
- **時間設定**: プリセット（3:00 / 4:00 / 4:30）+ 自由入力（M:SS形式、Enterで確定、最大10:00）
- **スナップ**: 1秒 / 0.1秒 切替
- **レイヤー**: 1〜4個まで追加・削除可能
  - 削除時にスキルが配置されていたら確認ダイアログ表示
  - 奇数・偶数レイヤーで背景色を変えて区別
- **ズーム**: Ctrl+スクロール（非passive event listenerで登録し、ブラウザズームと競合回避）
- **左右余白**: 両端に60pxのパディング（端のスキルアイコンが切れないように）
- **Canvas目盛り**: 10秒ごとにラベル付き太線、1秒ごとに中線、0.1秒ごとに細線
- **カーソル**: マウス位置に縦線＋時間ラベル表示
- **コストグラフ（CostRuler）**: 時間目盛りとレイヤーの間にCanvas描画
  - 残りコストの推移を折れ線＋塗りつぶしで表示
  - 1コスト単位で目盛り、上限値に橙色破線
  - コスト不足区間は赤色で表示

### タイムラインアイテム
- 顔アイコン（48×48px）＋コスト/時間テキスト
- **表示切替**: デフォルトはコスト表示（`XX.X`、0.1刻み切り捨て）、クリックで時間表示（`M:SS.mmm`）に切替
  - アイテムごとに個別切替（`useTimeDisplay`フラグ）
  - コスト不足時は赤色表示（`.overrun`クラス）
- **配置**: 使用モードの生徒をドラッグ&ドロップでレイヤーに配置（スナップ付き）
- **移動**: 左クリックドラッグで時間方向＋レイヤー間を移動可能
- **削除**: 右クリック
- **ドラッグ中**: 画面中央に大きく時間表示
- **ドロップ時**: レイヤーがハイライト表示
- **コスト調整**: +/-ボタン（0.5刻み）で手動コスト増減
- **対象指定**: Shift+クリックでEXスキルの対象スロットを選択、小アイコン表示

### 吹き出しコメント
- スキルアイテムをダブルクリック → コメント入力モーダル（最大30文字、Enter確定 / Escape取消）
- 吹き出しはレイヤー上部の専用エリアに黄色い吹き出しとして表示
- 破線でアイテムまで接続線を引く
- 重なる場合は自動的に行を分けて段積み配置
- コメント付きアイテムは黄色枠＋右上にドット表示
- 吹き出しを右クリックで削除

### 共有パネル（下部）
- **圧縮共有コード**: deflate圧縮 + 短縮キー名 + 画像URL除去で短いコードを生成
  - 新フォーマット: `Z:` プレフィックス + deflate圧縮Base64
  - 旧フォーマット（プレフィックスなしBase64）もインポート互換性あり
  - キャラ画像はインポート時にキャラ名からcharacters_st/sp.jsonで自動解決
- エクスポート: クリップボードに自動コピー（async処理、ローディング表示あり）
- インポート: テキストエリアにコードを貼り付けて復元
- 共有データに含まれる情報: 生徒スロット（名前・タイプ・位置）、アイテム配置（時間・レイヤー・コメント・コスト調整・対象・表示モード）、矢印、レイヤー数、総時間、スロットコスト設定

## スキルコストシステム

### コスト計算モデル
- 初期コスト: 0（戦闘開始=タイムライン最大時間から）
- **回復開始遅延**: 戦闘開始から2.033秒後に回復開始（`RECOVERY_DELAY_MS = 2033`）
- 時間経過で回復（コスト/秒）、スキル使用で減少
- コスト上限: 10 + 0.5 × (固有4チェック済みSP生徒数)
- 上限到達時は回復停止、スキル使用後に再開
- 手動コスト調整（`costAdjustment`）でスキル使用前にコストを増減可能

### コスト回復速度の計算
```
基礎回復力 = 各キャラ700 × 6人 = 4200
最終速度 = (合計回復力) × 乗数 / 10000
```

### 自動判定キャラ
- **チェリノSS +511**: チェリノ, チェリノ（温泉） → 合計回復力に+511
- **SP SS ×1.202** (重複不可、SPスロットのみ): ヒマリ, キサキ, ハルカ（正月）, シロコ（水着）, ミチル（ドレス）, ヒマリ（臨戦）, ニヤ
- **ホシノ（水着）EX**: +684回復力, 50秒間
- **ヒフミEX**: +1861回復力, 5秒間
- **セイアEX**: +378回復力, 15秒間

EXバフはスキル使用時に開始、持続時間後に終了。バフ期間中のみ回復速度が増加。

### コスト計算エンジン（`src/utils/costCalc.ts`）
- `calculateCostCap()`: コスト上限算出
- `calculateItemCosts()`: 全アイテムのコスト値＋オーバーラン判定 → `Map<string, ItemCostInfo>`
- `calculateCostTimeline()`: グラフ描画用キーポイント列 → `CostKeypoint[]`
- イベントベース処理: スキル使用・バフ開始・バフ終了を時間順に処理

## 座標変換

- `totalWidth = TIMELINE_PAD_LEFT + totalTimeS * zoomLevel + TIMELINE_PAD_RIGHT`
- `timeToX(ms) = totalWidth - TIMELINE_PAD_RIGHT - (ms / 1000) * zoomLevel`
- `xToTime(x) = (totalWidth - TIMELINE_PAD_RIGHT - x) / zoomLevel * 1000`

## 操作手順

1. 編集モード（初期状態）で生徒を登録（1人でもタイムラインに配置可能）
2. 各スロットのスキルコストを設定（デフォルト3）、SPスロットは固有4チェック
3. 「使用中」ボタンに切り替え
4. 生徒をドラッグしてタイムラインのレイヤーにドロップ
5. 配置したスキルは左クリックドラッグで移動、右クリックで削除
6. クリックでコスト⇔時間表示を切替、+/-ボタンでコスト手動調整
7. Shift+クリックでEX対象スロットを指定
8. ダブルクリックでコメント吹き出しを追加
9. 完成したら共有パネルでエクスポート

## コスト計算式一覧
・コスト回復の計算式（基本）
　コスト回復速度(コスト / 秒) = (全員のコスト回復力の合計) / 10000
　※総力戦6人設定可能
　※各生徒固有値700
　※全員のコスト回復力の合計 = 700 * 6

・コスト回復の計算式（スペシャル[ヒマリ・キサキ・ハルカ（正月）・シロコ（水着）・ミチル（ドレス）・ヒマリ（臨戦）・ニヤ]SS有[重複不可]）
　コスト回復速度(コスト / 秒)  = (全員のコスト回復力の合計) * 1.202 / 10000
　※コスト回復SS　20.2%増加

・コスト回復の計算式（チェリノSS有）
　コスト回復速度(コスト / 秒)  = (全員のコスト回復力の合計 + 511 + (0 or 146 or 292 or 438)) * 1.202 / 10000
　※チェリノSS10LV　コスト回復力を511増加
　※チェリノSS10LV　自身を除くレッドウィンター生徒一人につき追加でコスト回復力が146追加（最大3人）

・コスト回復の計算式(ホシノ（水着）EX)
　コスト回復速度(コスト / 秒)  = ((ホシノ（水着）を除いたコスト回復力の合計) + 700 + 684)* 1.202 / 10000
　※水着ホシノSS10LV　EX発動中(50秒間)自身のコスト回復力を684増加

・コスト回復の計算式(ヒフミEX)
　コスト回復速度(コスト / 秒)  = ((ヒフミを除いたコスト回復力の合計)  + 700 + 1861) / 10000
　※ヒフミSS10LV　EXスキル使用時5秒間コスト回復力を1861増加

・コスト回復の計算式(セイアEX)
　コスト回復速度(コスト / 秒)  = ((セイアを除いたコスト回復力の合計)  + 700 + 718) / 10000
　※セイアEX10LV　EXスキル使用時15秒間自身のコスト回復力を718増加

・スペシャル一人、ホシノ（水着）、セイア、ヒフミ、チェリノ、は重複可能、全て重複した場合は以下の計算式になる
　※6人でチェリノ・ホシノ（水着）・ヒフミ・セイアEX発動時の場合
　((700 + 511)(チェリノ)  + (700 + 684)(ホシノ（水着）EX効果中) + (700 + 1861)(ヒフミEX効果中) + (700 + 718)セイアEX発動中) 700 + 700) * 1.202(SP SS) / 10000

・該当生徒が設定されていた場合は自動的に計算式を修正する
　スキルレベルは全てMAXで想定
　生徒は途中で落ち事を想定しない
・コスト回復開始は戦闘開始から2.033秒後

### コスト計算式追加分
・コスト回復の計算式(カノエSS)
　コスト回復速度(コスト / 秒)  = ((コスト回復力の合計)  + 342 + (85 * X)) / 10000
　※カノエSS10LV　コスト回復力を342増加
　※防御タイプが重装甲のストライカーの味方一人につき85増加（自身は除外、最大三人）

・コスト回復の計算式(クルミSS)
　コスト回復速度(コスト / 秒)  = ((コスト回復力の合計)  + 183 + 854（戦術シールド発動中のみ) / 10000
　※クルミSS10LV　コスト回復力を183増加
　※クルミEX10LV　10秒間戦術シールドを発動
　※戦術シールドが維持されている間は追加でコスト回復力854増加

・コスト回復の計算式(スズミ（マジカル）)
　コスト回復速度(コスト / 秒)  = ((コスト回復力の合計)  + 690（EX発動から35秒) / 10000
　※スズミ（マジカル）SS10LV　EXスキル発動時、コスト回復力を690増加（35秒）
　※スズミ（マジカル）EX10LV　バフ効果（これを起点にSSが誘発）

・コスト回復の計算式(ノア（パジャマ）)
　コスト回復速度(コスト / 秒)  = ((コスト回復力の合計)  + 803（EX2回発動から30秒) / 10000

　※再びコスト5版でEXを2回打つとNSの効果が誘発する、これは繰り返される
