# ブルアカ TL作成支援ツール

ブルーアーカイブの総力戦・大決戦等でスキル使用タイミングを記録・共有するためのタイムラインツールです。

## URL

https://kur-3dcg.github.io/blue-archive-tl-tool/

## 主な機能
- 編成機能（通常モード: 6人 / 制約解除決戦モード: 10人）
- ドラッグ&ドロップでスキル使用タイミングを配置
- コスト自動計算（SS・EXバフ対応、重装甲・RW自動判定、ナギサ（水着）オーバーコスト対応）
- **ステージギミック対応**（コクマー バリスタ等のステージ固有効果をコスト計算・グラフに反映）
- **スキルキューモード**（スキル使用順バッジ・スキル順検証・ゲーム再現モード）
- **多言語対応**（日本語 / English / 中文(简) / 中文(繁) / 한국어）
- セーブ・ロード機能（30スロット）・**JSONファイル保存/読込**
- URL共有・TLテキスト出力・TL画像出力（透過PNG / 白背景PNG）
- **etcアイコン対応**（コピーアイコン等をEX対象指定に追加、テキスト・画像出力にも反映）
- コメント吹き出し（生徒依存/独立）・矢印接続
- テキストマーカー（M:SS形式で一括入力 → タイムラインにティール縦線表示）

## 操作

| 操作 | 動作 |
|------|------|
| クリック | コスト⇔時間 表示切替 |
| ダブルクリック | フリーコメント追加（TL上どこでも可） |
| 右クリック | 削除 |
| ドラッグ | 移動 |
| Shift+クリック | EX対象指定 |
| Ctrl+クリック | キャラ依存コメント追加 |
| Alt+クリック | 矢印接続 |
| Ctrl+ホイール | TL横移動 |

## 使い方

### 1. 生徒を登録する
- 起動時は「編成中」モードです。空のスロットをクリックして生徒を選択してください
- 通常モードはSTRIKER 4枠、SPECIAL 2枠（計6人）まで、制約解除決戦モードはSTRIKER 6枠、SPECIAL 4枠（計10人）まで登録できます
- SPスロットの「固有4」チェックを入れるとコスト上限が+0.5されます
- 「固有2」チェック（全スロット、デフォルトON）でバフ効果時間が×1.19される生徒に適用されます（コスト回復EXバフ・バフ時間バー両方に反映）
- SPスロットの「固有4」チェックは固有2以上を意味するため、バフ効果時間×1.19も同時に適用されます
- 重装甲・レッドウィンター人数は編成から自動判定されます

### 2. スキルをタイムラインに配置する
- 「TL作成中」ボタンを押してモードを切り替えます
- 生徒アイコンをドラッグしてタイムラインのレイヤーにドロップします
- 配置されたスキルにはコストが自動計算されて表示されます

### 3. スキルを編集する
- **移動**: 配置済みスキルを左クリックドラッグで移動
- **削除**: 右クリックで削除
- **コスト/時間 切替**: クリックで表示をコスト⇔時間に切替
- **コスト手動調整**: +/- ボタンでスキルコストを個別に増減
- **EX対象指定**: Shift+クリックでメニューを開き対象を選択
  - etcアイコン（コピーアイコン等）: 左上に小アイコン表示。リオが編成中のときのみ表示
  - 生徒スロット: 右上に小アイコン表示
- **フリーコメント**: TLレイヤー上をダブルクリックでフリーコメント（緑バブル）を追加。アイコン上でダブルクリックするとそのスキルの時間に配置
- **キャラ依存コメント**: スキルアイコンを Ctrl+クリックでコメント吹き出し（黄色）を追加（吹き出しを右クリックで削除）
- **移動ロック**: 🔓/🔒ボタンでスキルアイコンのドラッグ移動を禁止できます。クリック系の操作（コメント追加・表示切替・コスト調整等）は引き続き有効です

### 4. ステージギミックを設定する
- コメントボタン横の「ギミック」ボタンでパネルを開きます
- プリセット（コクマー バリスタ等）または「カスタム」を選択し、発動時間（M:SS形式）を入力して「追加」
  - カスタムの場合は名前・回復力増加量・効果時間（秒）も入力
- 設定したギミックはコストグラフ上部に橙色の帯として表示され、コスト計算に反映されます
- 共有コードに含まれるため URL 共有後もギミック設定が維持されます

### 5. 独立コメント・テキストマーカーを配置する
- **ダブルクリック追加**: TLレイヤー上をダブルクリック → その位置にフリーコメント（緑バブル）を追加
- **ドラッグ追加**: 「TL作成中」モードで、コメントボタンをドラッグ → タイムラインにドロップ
  - 緑色の吹き出しとして専用エリアに表示
  - ドラッグで移動、右クリックで削除、ダブルクリックで編集
- **テキスト入力**: コメントボタン下の「テキスト」ボタンでパネルを開く
  - `3:30.00 テキスト` の形式で1行1マーカーを入力
  - タイムライン上にティール色の縦線＋ラベルとして表示
  - ドラッグ追加したコメントも自動的にパネルに反映される

### 6. タイムラインの設定
- **時間**: プリセット（3:00 / 4:00 / 4:30 / 5:00）または自由入力（M:SS形式、Enterで確定）
- **スナップ**: ボタンクリックで1秒/0.1秒を切替
- **レイヤー**: 1〜4個まで変更可能
- **ビューポート移動**: 左右の◀▶ボタンで30秒刻みでスクロール
- **目標時間**: 「目標」欄にM:SS形式で入力するとタイムラインに赤い縦線を表示

### 7. 共有・出力する
- **URL出力**: URLをクリップボードにコピーします。Twitter等に貼り付けてそのまま共有できます
- **TL出力**: TLをTSV形式でクリップボードにコピーします（スプレッドシートに貼り付け可能）
- **画像出力（透過）**: TLを透過PNG画像としてダウンロードします（生徒アイコン・EX対象アイコン付き）
- **画像出力（白）**: TLを白背景PNG画像としてダウンロードします
- **JSON保存**: 現在のTLをJSONファイルとして保存します（バックアップや手元保存に便利）
- **JSONから読込**: 保存したJSONファイルを選択して読み込みます
- **インポート**: 共有URLまたはコードを貼り付けて復元します

### 8. セーブ・ロード
- ページをリロードしても編成・TLは自動的に復元されます
- 「セーブ」ボタンで最大30スロットに保存できます
- 「ロード」ボタンで過去のデータを読み込めます

### 9. スキルキューモード
- **スキル使用順バッジ**: 「編成中」モードで、アクティブスロット数を超えるキャラが登録されると各スロットの左上に順番バッジが表示されます
  - バッジをクリックするとピッカーが開き、位置を直接選択できます
  - 通常モード（6人）: アクティブ枠 1〜3 はカラー（緑/黄/青）、4〜6 はグレー
  - 制約解除決戦モード（10人）: アクティブ枠 1〜5 はカラー（緑/黄/青/橙/紫）、6〜10 はグレー
- **スキル順検証**: タイムラインの「スキル順」ボタンをONにすると、アクティブスロット外から使われたスキルを赤オーバーレイで表示します
- **ゲーム再現モード**: 「TL作成中」に切り替えた状態で「ゲーム再現」ボタンをONにすると、現在アクティブなスキルスロット（通常3枠・制約解除決戦5枠）を A'/B'/C'... として表示します
  - TLに配置済みのスキルから現在のキュー状態を自動シミュレーション
  - 各スロットをドラッグしてタイムラインに配置できます

### 10. 矢印接続
- **ショートカット**: スキルアイコンを Alt+クリックして別のスキルを Alt+クリックで接続（矢印ボタン不要）
- 「矢印」ボタンをONにすると矢印モードになり、通常クリックで接続できます
- 矢印を右クリックで削除

### 11. 言語切替
- ヘッダー右上の「Language」セレクタで表示言語を切り替えられます
- 対応言語: 日本語 / English / 中文(简) / 中文(繁) / 한국어
- 設定はブラウザに保存されます（次回起動時も維持）
- 生徒名も各言語名で表示されます（グローバル版未実装キャラは日本語名にフォールバック）
- 非日本語モードでは生徒検索が各言語名でも動作します
