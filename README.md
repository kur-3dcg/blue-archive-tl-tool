# ブルアカ TL作成支援ツール

ブルーアーカイブの総力戦・大決戦等でスキル使用タイミングを記録・共有するためのタイムラインツールです。

## URL

https://kur-3dcg.github.io/blue-archive-tl-tool/

## 主な機能

- ストライカー4枠 + スペシャル2枠の生徒登録
- ドラッグ&ドロップでスキル使用タイミングを配置
- コスト自動計算（SS・EXバフ対応）
- URL共有（Twitter等での共有に対応）
- コメント吹き出し・矢印接続

## 操作

| 操作 | 動作 |
|------|------|
| クリック | 選択 |
| ダブルクリック | コメント追加 |
| 右クリック | 削除 |
| ドラッグ | 移動 |
| Shift+クリック | EX対象指定 |
| Ctrl+クリック | 矢印接続 |

## 使い方

### 1. 生徒を登録する
- 起動時は「編集モード」です。空のスロットをクリックして生徒を選択してください
- STRIKER 4枠、SPECIAL 2枠まで登録できます
- 各スロット下の「コスト」欄でEXスキルコストを設定できます（デフォルト3）
- SPスロットの「固有4」チェックを入れるとコスト上限が+0.5されます

### 2. スキルをタイムラインに配置する
- 「使用中」ボタンを押してモードを切り替えます
- 生徒アイコンをドラッグしてタイムラインのレイヤーにドロップします
- 配置されたスキルにはコストが自動計算されて表示されます

### 3. スキルを編集する
- **移動**: 配置済みスキルを左クリックドラッグで移動
- **削除**: 右クリックで削除
- **コスト/時間 切替**: クリックで表示をコスト⇔時間に切替
- **コスト手動調整**: +/- ボタンでスキルコストを個別に増減
- **EX対象指定**: Shift+クリックで対象スロットを選択（小アイコンで表示）
- **コメント**: ダブルクリックでコメント吹き出しを追加（吹き出しを右クリックで削除）

### 4. タイムラインの設定
- **時間**: プリセット（3:00 / 4:00 / 4:30）または自由入力（M:SS形式、Enterで確定）
- **スナップ**: ボタンクリックで1秒/0.1秒を切替
- **レイヤー**: 1〜4個まで変更可能
- **ビューポート移動**: 左右の◀▶ボタンで30秒刻みでスクロール
- **目標時間**: 「目標」欄にM:SS形式で入力するとタイムラインに赤い縦線を表示
- **重装甲 / RW**: コスト計算に影響する重装甲・レッドウィンター生徒の人数を設定

### 5. 共有する
- **URL**: URLをクリップボードにコピーします。Twitter等に貼り付けてそのまま共有できます
- **Code**: 共有コードをクリップボードにコピーします
- **Import**: 共有URL またはコードを貼り付けて「Import」ボタンで復元します

### 6. 矢印接続
- 「矢印」ボタンをONにすると、スキル間に矢印を引けます
- スキルアイコンからドラッグして別のスキルにドロップで接続
- 矢印を右クリックで削除
