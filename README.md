# ブルアカ TL作成支援ツール

ブルーアーカイブの総力戦・大決戦等でスキル使用タイミングを記録・共有するためのタイムラインツールです。

## URL

https://kur-3dcg.github.io/blue-archive-tl-tool/

## 主な機能
- 編成機能（通常モード・制約解除決戦モード）
- ドラッグ&ドロップでスキル使用タイミングを配置
- コスト自動計算（SS・EXバフ対応、重装甲・RW自動判定、ナギサ（水着）オーバーコスト対応）
- セーブ・ロード機能（30スロット）
- URL共有・TLテキスト出力・TL画像出力（透過PNG / 白背景PNG）
- コメント吹き出し（生徒依存/独立）・矢印接続
- テキストマーカー（M:SS形式で一括入力 → タイムラインにティール縦線表示）

## 操作

| 操作 | 動作 |
|------|------|
| クリック | コスト⇔時間 表示切替 |
| ダブルクリック | コメント追加 |
| 右クリック | 削除 |
| ドラッグ | 移動 |
| Shift+クリック | EX対象指定 |
| Ctrl+クリック | 矢印接続 |

## 使い方

### 1. 生徒を登録する
- 起動時は「編成中」モードです。空のスロットをクリックして生徒を選択してください
- 通常モードはSTRIKER 4枠、SPECIAL 2枠まで、制約解除決戦モードはSTRIKER 6枠、SPECIAL 4枠まで登録できます
- SPスロットの「固有4」チェックを入れるとコスト上限が+0.5されます
- 「固有2」チェック（全スロット、デフォルトON）でバフ効果時間が×1.19される生徒に適用されます
- 重装甲・レッドウィンター人数は編成から自動判定されます

### 2. スキルをタイムラインに配置する
- 「TL作成中」ボタンを押してモードを切り替えます
- 生徒アイコンをドラッグしてタイムラインのレイヤーにドロップします
- 配置されたスキルにはコストが自動計算されて表示されます

### 3. スキルを編集する
- **移動**: 配置済みスキルを左クリックドラッグで移動
- **削除**: 右クリックで削除
- **コスト/時間 切替**: クリックで表示をコスト⇔時間に切替
- **コスト手動調整**: +/- ボタンでスキルコストを個別に増減
- **EX対象指定**: Shift+クリックで対象スロットを選択（小アイコンで右上表示）
- **コメント**: ダブルクリックでコメント吹き出しを追加（吹き出しを右クリックで削除）

### 4. 独立コメント・テキストマーカーを配置する
- **ドラッグ追加**: 「TL作成中」モードで、コメントボタンをドラッグ → タイムラインにドロップ
  - 緑色の吹き出しとして専用エリアに表示
  - ドラッグで移動、右クリックで削除、ダブルクリックで編集
- **テキスト入力**: コメントボタン下の「テキスト」ボタンでパネルを開く
  - `3:30.00 テキスト` の形式で1行1マーカーを入力
  - タイムライン上にティール色の縦線＋ラベルとして表示
  - ドラッグ追加したコメントも自動的にパネルに反映される

### 5. タイムラインの設定
- **時間**: プリセット（3:00 / 4:00 / 4:30 / 5:00）または自由入力（M:SS形式、Enterで確定）
- **スナップ**: ボタンクリックで1秒/0.1秒を切替
- **レイヤー**: 1〜4個まで変更可能
- **ビューポート移動**: 左右の◀▶ボタンで30秒刻みでスクロール
- **目標時間**: 「目標」欄にM:SS形式で入力するとタイムラインに赤い縦線を表示

### 6. 共有・出力する
- **URL出力**: URLをクリップボードにコピーします。Twitter等に貼り付けてそのまま共有できます
- **TL出力**: TLをTSV形式でクリップボードにコピーします（スプレッドシートに貼り付け可能）
- **画像出力（透過）**: TLを透過PNG画像としてダウンロードします（生徒アイコン・EX対象アイコン付き）
- **画像出力（白）**: TLを白背景PNG画像としてダウンロードします
- **インポート**: 共有URLまたはコードを貼り付けて復元します

### 7. セーブ・ロード
- ページをリロードしても編成・TLは自動的に復元されます
- 「セーブ」ボタンで最大30スロットに保存できます
- 「ロード」ボタンで過去のデータを読み込めます

### 8. 矢印接続
- 「矢印」ボタンをONにすると、スキル間に矢印を引けます
- スキルアイコンをクリックして別のスキルをクリックで接続
- 矢印を右クリックで削除
